# でかニュージャージー

アズールレーン Wiki のデータを基に、各艦船（艦娘）の設定情報を確認できます。

## アズールレーン

以下はアズールレーンの検索コマンドです（! をトリガーとして使用できます）：
※現在、コマンドは 中国語のみ に対応しています。
!help -> ヘルプコマンド

!碧蓝航线 -> ルートコマンド

!碧蓝航线 舰娘 -> 艦船（艦娘）検索コマンド

!碧蓝航线 舰娘 新泽西 -> 特定艦船（艦娘）検索コマンド（「新泽西」に艦船の中国語名を入力）

!碧蓝航线 装备 -> 装備検索コマンド

!碧蓝航线 装备 双联100mm98式高射炮改#T0 -> 特定装備検索コマンド（「双联100mm98式高射炮改#T0」に装備の中国語名を入力）

!碧蓝航线 月度 -> 月間チャレンジ検索コマンド

!碧蓝航线 井号 -> ハッシュタグランキング検索コマンド

現在対応しているのは、メインストーリー、オープンワールド、装備、金部品、研究ランキング の検索のみです。

!碧蓝航线 井号 装备 -> このコマンド内の「装备」は、
メインストーリー（主线）、オープンワールド（大世界）、金部品（金部件）、研究ランキング（研发榜）に置き換えることができます。
